## 一、股市的基本概念

1. 股市的基本结构：
   1. A股（**T+1**）：
      - 主要交易所：
        - 上交所（上海证券交易所）
        - 深交所（深圳证券交易所）
      - 板块分类：
        - 主板：成熟企业（上交所+深交所；上市门槛高，一般是成熟大企业；**涨跌幅10%，波动较小，机构参与更多**）
        - 创业板：成长型企业（深交所；集中在医药、新能源、新材料、科技领域；涨跌幅20%；**情绪驱动明显——牛市涨得快，熊市跌得猛；受大众情绪影响大**）
        - 科创板：硬科技创新企业（上交所；国家重点扶持科技企业，主要集中在AI、半导体、生物科技、高端制造；**上市前5天无涨跌停，后续涨跌幅20%；波动明显，机构参与度高，受科技创新周期驱动**）
   2. 港股（**T+0，无涨跌停限制**）：港交所。
   3. 美股（机构占主导，长期投资更常见）：
      - 交易所：
        - 纽交所（纽约证券交易所）
        - 纳斯达克证券交易所

## 二、股市的核心指标

1. 价格类指标（K线相关）：
   - 开盘价
   - 收盘价
   - 最高价
   - 最低价

2. 成交量（反映资金的活跃程度）：
   - **价升量增才健康**
   - 1手 = 100股
   - $$
     换手率 = \frac{成交量}{流通股本}
     $$
   - 换手率更能反映资金博弈强度，可以用于研究个股资金博弈，识别短线活跃股。
   - 换手率参考区间：

     | 换手率 | 含义 |
     | ------ | ---- |
     | <1% | 冷清 |
     | 1–3% | 正常 |
     | 5–10% | 活跃 |
     | >20% | 激烈博弈 |
     | >40% | 情绪极端 |

     但需要结合市值综合判断。

   - 量化里怎么用？

     常见做法：
     1. 过滤换手率过低股票（流动性风险）
     2. 研究高换手突破是否延续
     3. 构造“换手率因子”

     比如：
     - 低换手率 + 上涨 → 稳定趋势
     - 高换手率 + 下跌 → 恐慌释放

     换手率其实在衡量：
     > 筹码分布是否在转移

     高换手意味着：
     - 原有持仓被洗牌
     - 成本区间重构

     这会改变未来压力位和支撑位。

3. 市值
   - 总市值 = 股价 × 总股本
   - 流通市值 = 可交易股本 × 股价

4. 估值指标
   - PE（市盈率）：股价 / 每股盈利
     - PE高 = 市场预期高
     - PE低 = 市场预期较低或估值相对便宜
   - PB（市净率）：股价 / 每股净资产

5. ROE（净资产收益率）：企业赚钱能力。作为股东，我今年投入了X元，年底这家公司能给我带来Y元收益（净利润），则这家公司当年的 $ROE = \frac{Y}{X}$。

## 四、补充理解

### 1. 关于“仓”的理解

- 建仓 = 第一次买。
- 满仓 = 钱全买光，不留现金（将所有资金都用来投资，没有备用金——当股价稍跌时没有能力补仓）。
- 加杠杆 = 借钱炒股，新手禁区。

### 2. 关于高频 / 中频 / 低频（新手应做中低频）

- 高频：天天买卖、频繁操作。
- 中频：几周～几个月买卖一次。
- 低频：拿很久，几个月～几年才动一次。

新手不能做高频（经典高频操作：虽然只关注一支股票，但每天频繁进行加减仓操作），手续费占比非常大：

- 手续费会吃掉很多利润：买一次、卖一次都要钱，天天动 → 一年下来手续费惊人。
- 短线涨跌随机性强：今天涨、明天跌，没人能精准预测；每天操作等于天天猜硬币。
- 心态容易崩：每天盯盘，心情跟着K线走，赚了飘、亏了慌，越做越错。
- 90% 新手高频操作最后都亏损；真正赚钱的人大部分是低频、拿得住的人。

新手应该只看 1～2 只股票，但操作频率要改成：

- 建仓后，不轻易动。
- 一周看 2～3 次就够。
- 不到止损不止损，不到止盈不止盈。
- 不每天补仓、不每天减持。

这叫：低频观察 + 中频操作——这才是新手能赚钱的节奏。

### 3. 量化投资的风险

- 市场流动性风险：同质化量化交易策略导致的市场性共振与系统性风险。
- 策略失效风险：最严重因素之一，且失效时间不可预知。

90年代是量化投资发展的黄金期，量化投资应用于市场中的各种投资工具，各大理论全面发展。著名的资产资本定价模式、市场有效性理论、期权定价理论、套利理论也在这一时间产生，这些理论为量化投资策略成为系统有效的策略奠定了科学理论基础。
